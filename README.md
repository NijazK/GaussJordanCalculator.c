# Guass-Jordan Calculator  
  * Puzzle to solve any square matrix using Reduce row Echelon
  * Written in C
  * Reduce-row echelon

## Instructions
  
1. Input the square matrix
    ex: 3 
2. Augment the matrix
    ex: A[1][1]: 5
    
        A[1][2]: 7
        
        A[1][3]: 3
        
        A[1][4]: 5
        
        A[2][1]:-5
        
        A[2][2]:-4
        
        A[2][3]: 8
        
        A[2][4]: 10
        
        A[3][1]: 3
        
        A[3][2]:-2
        
        A[3][3]: 1
        
        A[3][4]: 2
        
    Results:
      X1 = 0.2006075
      X2 = -0.01597
      X3 = 1.36778
